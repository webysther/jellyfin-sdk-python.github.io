# DatabaseLockingBehaviorTypes

Defines all possible methods for locking database access for concurrent queries.

## Enum

* `NOLOCK` (value: `'NoLock'`)

* `PESSIMISTIC` (value: `'Pessimistic'`)

* `OPTIMISTIC` (value: `'Optimistic'`)
